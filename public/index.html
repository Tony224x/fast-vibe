Quick guide

Layout — drag to split

  • Drag a pane header onto another pane to see 4 drop zones (top / right / bottom / left) and a center zone.
  • Drop on top / right / bottom / left to split the target in that direction.
  • Drop on the center to stack the panes as tabs — switch with the ◄ ► arrows in the header.
  • Drag the splitter bars between panes to resize. Layout is saved automatically.

Groups

  • Ctrl-click two or more pane headers to multi-select (orange outline appears).
  • A floating Group button appears at the bottom of the screen — click it to wrap the selected panes.
  • The group gets a colored border and dot. Double-click the name to rename it.
  • Group header buttons: collapse, broadcast (send input to all panes in group), ungroup.
  • Selection is only valid if the panes form a contiguous subtree — otherwise the toolbar shakes.

Pane actions

  • Verify — ask the worker to review its own output.
  • Copy — copy current pane output to clipboard.
  • Compact / Clear — manage Claude context.
  • Restart — relaunch the worker.
  • Expand (top-right arrows) — fullscreen the pane.

Shortcuts

  • Ctrl + Shift + F — toggle Zen mode (hides launch bar + sidebar).
  • ? — open this help.
  • Esc — close modal / clear group selection.
